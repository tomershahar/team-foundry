# team-foundry Coach Playbook

This is the Clearline team's coach. It runs the same playbook as the standard team-foundry coach — see the [team-foundry repo](https://github.com/tomershahar/team-foundry) for the full behavior list.

This file is intentionally short in the example repo. In a real team-foundry, this file is generated by `npx create-team-foundry` and contains the full coach playbook (~1,400 lines).

## Quick reference

To see the full coach in action, run `npx create-team-foundry` in your own repo. The generated `.team-foundry/coach.md` will contain:

- 17 coaching behaviors (B1–B17)
- Inline, explicit, and scheduled activation modes
- Conversation-as-update protocol
- Nudge memory and context priority
- Quarterly retrospective

## Activation

Say any of these to Claude Code:

| What to say | What happens |
|---|---|
| "let's do a team-foundry review" | Full audit of all Clearline files |
| "review our outcomes" | Targeted review of outcomes.md |
| "what's missing from team-foundry?" | Lists gaps across all files |
| "run the weekly team-foundry review" | Top 3 findings this week |
